# Super Market - E-Commerce Web Application

A full-featured online supermarket application built with Next.js, React, TypeScript, and MongoDB.

## Features

### Customer Features
- Product browsing by dynamic categories
- Real-time search
- Shopping cart with quantity management
- Razorpay payment integration
- Address management
- Order tracking
- User authentication

### Admin Features
- Admin dashboard
- Product management (CRUD operations)
- Dynamic category creation
- Order management
- Session persistence
- Stock tracking

## Tech Stack

- Next.js 16.1.4 + React 19.2.3 + TypeScript
- MongoDB + Mongoose
- Razorpay Payments
- Tailwind CSS

## Installation

1. Clone: git clone https://github.com/YOUR_USERNAME/super-market.git
2. Install: npm install
3. Create .env.local with MongoDB and Razorpay credentials
4. Run: npm run dev
5. Open: http://localhost:3000

## Admin Setup

Create admin: node create-admin.js
